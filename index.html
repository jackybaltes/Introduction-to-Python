Fizz Buzz Problem

FizzBuzz is a simple programming problem, which is based on an Australian childrens game.

The object of the game is to teach children about multiplication

The children count from 1 to 100. So 1,2,... with the exception:

  • if a number is a multiple of 3, then you say "Fizz" instead
  • if a number is a multiple of 5, then you say "Buzz" instead
  • if a number is a multiple of 3 and 5, then you say "FizzBuzz" instead
# Buggy implementation of FizzBuzz

def fizzBuzz(N):
  for i in range(1,N+1):
    if ( i % 3 == 0 ):
      print("Say", "Fizz")
    elif (i % 5 == 0 ):
      print("Say", "Buzz")
    elif ( ( i % 3 == 0) and ( i % 5 == 0) ):
      print("Say", "FizzBuzz")
    else:
      print("Say", i)

def maxOdd(lst):
  "returns max. odd number in a list"
  m = None
  for e in lst:
    if ( e % 2 == 1 ):
      if m is None or ( e > m ):
        m = e
  return m

import random

def find2ndLargest( lst ):
  m1 = None
  m2 = None
  for el in lst:
    if m1 is None or el > m1:
      m2 = m1
      m1 = el
    elif m2 is None or el > m2:
      m2 = el
  return m2
